OKFocus Video

OKFocus video is a JQuery plugin that turns videos on YouTube or Vimeo into beautiful full-screen backgrounds, like the one on this page from Gatekeeper.

Using OKVideo is stupid simple. Just add the script and use the following code : 

$(function(){
   $.okvideo({ source: 'YOUR AWESOME YOUTUBE OR VIMEO LINK HERE' }) 
});

Additional params you can pass are as follow : 

		

		

		option
		      description
		      default
		    

		

		source
		       an id or url from vimeo or youtube
		       null
		    

		

		disablekeyControl
		       enable or disable key control (youtube videos only)
		       1
		    

		

		captions
		      enable or disable captions (youtube videos only)
		       0
		    

		

		loop
		      loop the video
		        1
